Usage Guidelines

Best practices for implementing the Arctic Guardian logo across all media.

Do ✓

  • Use approved logo files only
  • Maintain minimum clear space
  • Use on solid, clean backgrounds
  • Scale proportionally
  • Use approved color variations
  • Ensure proper contrast
  • Follow minimum size guidelines

Don't ✗

  • Rotate or tilt the logo
  • Stretch or distort proportions
  • Change colors arbitrarily
  • Add effects (shadows, glows)
  • Place on busy backgrounds
  • Recreate or modify elements
  • Use below minimum size
